# opentie app

opentie のバックエンド

## 何

学園祭実行委員会と参加者をつなぐソフトウェア。

## 設計

バックエンドは[これ](https://github.com/opentie/opentie-app)、フロントエンドは[opentie-console](https://github.com/opentie/opentie-console)。

バックエンドは JSON を吐く API サーバーとして動く。DB は PostgreSQL を使いたい。
